Изначальный кот

Примеры стилизации шаблона и отдельных экземпляров

  • Просто котик
    Просто иконка, загруженная как шаблон и инстациированная элементом use
  • Оранжевохвостые коты
    Стилизуем исходный шаблон; применяется ко всем инстансам
  • Разнохвостые коты
    Частям исходного шаблона явно указываем брать цвет у родительских элементов
  • Погладь кота!
    Подсветка иконки при ховере; появление ранее скрытых элементов рисунка при выборе
  • Мейн-кун - это тема
    Темирование иконки на уровне страницы, оставляя возможность кастомизации
  • Кошка о двух цветах
    Изменение двух цветов отдельного экземпляра иконки при помощи хакотрюка с currentColor
  • Кот в нокауте
    Пример использования с библиотекой Knockout
  • Отзывчивые коты
    Использовине иконок в responsive-разметке
  • Выбери кота!
    Появление ранее скрытых элементов рисунка при выборе
  • Котов нет, просто фигуры
    Пример сброса цветов с помощью CSS
  • Сотни котов
    Множество иконок (стилизованных) - для оценки производительности
  • Сотни иконок
    Множество разных иконок (стилизованных) - для ещё большей оценки производительности
